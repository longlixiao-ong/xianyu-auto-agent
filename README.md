# 闲鱼自动客服助手

基于大模型的闲鱼自动回复机器人，支持智能对话、卡密自动发货、人工接管等功能。

> 本项目主要用于个人闲鱼账号自动客服场景，当前仓库提供核心代码和管理后台实现。由于 Cookie、模型密钥、提示词、商品数据、卡密数据均属于个人私有配置，仓库不会提供完整运行环境。其他用户如需使用，请根据自身账号环境自行补充 `.env`、提示词和商品配置。

## 核心功能

- **智能自动回复**：多层意图识别（关键词→正则→大模型），自动回复买家咨询
- **卡密自动发货**：虚拟商品自动发货，支持库存模式和固定内容模式
- **图片识别**：自动识别买家发送的图片（截图、报错、商品图），结合上下文回复
- **人工接管**：卖家发送特定关键词切换人工/自动模式
- **辱骂拦截**：检测买家辱骂消息，自动转人工处理
- **管理后台**：Web 管理界面，配置模型、查看日志、管理卡密、查看消息记录

## 环境要求

- Python 3.10+
- 支持 OpenAI 兼容接口的大模型（如小米 MiMo、通义千问等）

## 安装依赖

```bash
pip install websockets openai loguru python-dotenv httpx waitress
```

## 快速开始

### 1. 配置 .env 文件

在 `src/` 目录下创建 `.env` 文件：

```env
# 必填：大模型 API 密钥
API_KEY=your_api_key_here

# 必填：闲鱼 Cookie（从浏览器开发者工具获取）
COOKIES_STR=your_cookie_here

# 模型配置（单一模式）
MODEL_BASE_URL=https://your_model_api.com/v1
MODEL_NAME=your_model_name

# 管理后台访问令牌（用于保护写操作）
ADMIN_API_TOKEN=your_secure_token
```

### 2. 启动项目

```bash
cd src
python main.py
```

### 3. 访问管理后台

启动后访问 `http://127.0.0.1:18061`，使用 `ADMIN_API_TOKEN` 进行写操作。

## 功能说明

### 自动客服

- 自动识别买家意图（议价、技术咨询、通用问题）
- 支持图片消息识别和回复
- 检测到购买信号自动转人工
- 检测到辱骂消息自动转人工
- 支持模拟人工输入延迟

### 卡密管理

- **库存模式**：批量导入卡密，每次发货取出一张
- **固定内容模式**：所有买家收到相同内容（如网盘链接）
- 发货记录可查询，支持发货状态追踪

## 常见问题

### 风控处理

如果日志显示"风控拦截"或"Cookie 无效"：

1. 在浏览器中登录闲鱼网页版
2. 打开开发者工具 → Network → 复制 Cookie
3. 在管理后台「运营操作」页面更新 Cookie
4. 点击「启动客服」重新连接

### 商品列表为空

1. 在管理后台「运营操作」页面点击「刷新商品列表」
2. 或设置环境变量 `MY_ITEMS_SNAPSHOT_ON_START=True` 启动时自动加载

### 模型切换

在管理后台「模型」页面修改：

- **单一模型模式**：文字和图片使用同一个多模态模型
- **分模型模式**：文字和图片分别配置不同模型

修改后点击「保存并热切换」即时生效。

## 注意事项

### Cookie 安全

- Cookie 包含登录凭证，请勿泄露
- 不要将 `.env` 文件提交到代码仓库
- 定期更新 Cookie 避免过期

### 管理后台安全

- 后台默认绑定 `127.0.0.1`，仅本机可访问
- **不要将后台端口暴露到公网**，如需远程访问请使用 SSH 隧道
- 首次访问需输入 `ADMIN_API_TOKEN` 进行鉴权
- Token 存储在浏览器 `localStorage` 中，不会传输到后端页面

### 数据备份

重要数据文件位于 `src/data/` 目录：

- `chat_history.db`：聊天记录和卡密数据
- `dev_my_items.json`：商品列表快照

建议定期备份 `data/` 目录。
